# OpenPlumes
OpenPlumes is an open-source platform for probabilistic groundwater plume modeling, delivered through a QGIS plugin.

The project aims to combine Gaussian Processes, hydrogeologic knowledge, and uncertainty quantification into an accessible, open-source toolkit for groundwater professionals. While the initial focus is on plume interpolation and visualization, my hope is that the long-term vision is to provide advanced capabilities for physics-aware modeling, uncertainty analysis, monitoring network optimization, and future research in hybrid physics and machine learning approaches.
